# styled-strong-no-need

A group of stickers and posters with interaction design using Canvas API

## Overview

Interactive canvas-based animation system featuring animated stickers and posters with smooth mouse/touch interactions. Built with pure JavaScript and HTML5 Canvas API.

1. Open `index.html` in a modern web browser
2. Move your mouse to interact with elements
3. Click or tap to create new animated particles
